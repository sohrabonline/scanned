# scanned

Flutter plugin for scan barcode an QR in iOS and Android.

## Supported Platforms

| Platform | Check Connectivity | Listen for Changes |
| :------: |:------------------:| :----------------: |
| Android  |         ✅          |         ✅         |
|   iOS    |         ✅          |         ✅         |
|  macOS   |         ❌          |         ❌         |
|  Linux   |         ❌          |         ❌         |
| Windows  |         ❌          |         ❌         |
|   Web    |         ❌          |         ❌         |
